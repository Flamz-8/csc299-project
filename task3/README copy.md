# Task2 Study Planner App

## Overview
Task3 is an improved study planner application designed to help students manage their study sessions, tasks, and schedules effectively. It features a timer for tracking study sessions, a calendar for scheduling tasks, and a task manager for organizing study-related activities.

## Features
- **Timer**: Manage countdowns and track time spent on study sessions.
- **Calendar**: Schedule and display study sessions and tasks.
- **Task Manager**: Add, update, and delete tasks efficiently.
- **Study Tracker**: Integrate timer and task manager to monitor study progress.

## Installation
1. Clone the repository:
   ```
   git clone https://github.com/yourusername/task3.git
   ```
2. Navigate to the project directory:
   ```
   cd task3
   ```
3. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

## Usage
To run the application, execute the following command:
```
python src/main.py
```
Follow the on-screen instructions to manage your study sessions and tasks.

## Testing
To run the unit tests, use the following command:
```
pytest tests/
```
